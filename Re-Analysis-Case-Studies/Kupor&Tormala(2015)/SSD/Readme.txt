1. Subdirectory Mplus:

The Subdirectories "Mediation" and "Statistical Power" contain Mplus input files, 
SSD, and output files for the mediation models and statistical power estimations in the Appendix of
Meaningful Meaningful Analysis

2. Subdirectory R:

The subdirectory R contains the code and output for the mediation models and statistical power estimations in the Appendix of
Meaningful Meaningful Analysis


Note: Raw data to conduct these mediation analyses are available from the original authors (Kupor and Tormala 2015)
