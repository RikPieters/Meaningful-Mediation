Code to estimate mediation models on Summary Statistics Data (SSD)
Raw data to conduct mediation analyses are available from the original authors (Kim and McGill 2011) 


1. Subdirectory Mplus
Summary Statistics Data (SSD) and Mplus input files to estimate 
Mediation Models (1 and 2), and discriminant validity for Kim and McGill (2011, Study 2), 
as reported in "Meaningful Mediation Analysis".

2. Subdirectory R
The subdirectory includes the following:
readme-txt with instructions
R code for Model 1 and Model 2 with the SDD included 
R-code file name.txt with the output
